## Day 2 — Making Votes Actually Work

Yesterday you built the skeleton: nodes start, timers fire, elections are triggered. But right now `RequestVote` always says no, and `startElection()` never actually contacts anyone. You have candidates that never campaign and voters that never vote.

Today you fix both sides of that conversation.

The flow you're building is:

```
candidate                          peer
    |                               |
    |--- RequestVote(term, me) ---->|
    |                               | checks: is your term current?
    |                               | checks: did I already vote?
    |                               | checks: is your log up to date?
    |<-- VoteGranted: true ---------|
    |
    | (collect majority)
    | → become leader
```

By the end of today, one node should win an election, become leader, and the test should pass. Let's go.

## Step 1 — Send RequestVote RPCs from startElection() (25 min)

Replace your stub with this:

```go
func (rf *Raft) startElection() {
    rf.state = "candidate"
    rf.currentTerm++
    rf.votedFor = rf.me
    votes := 1
    term := rf.currentTerm

    for i := range rf.peers {
        if i == rf.me {
            continue
        }
        go func(peer int) {
            args := &RequestVoteArgs{
                Term:         term,
                CandidateId:  rf.me,
                LastLogIndex: len(rf.log) - 1,
                LastLogTerm:  rf.log[len(rf.log)-1].Term,
            }
            reply := &RequestVoteReply{}
            ok := rf.peers[peer].Call("Raft.RequestVote", args, reply)
            if !ok {
                return
            }

            rf.mu.Lock()
            defer rf.mu.Unlock()

            // If we've moved on, ignore stale reply
            if rf.currentTerm != term || rf.state != "candidate" {
                return
            }
            // Someone has a higher term — step down
            if reply.Term > rf.currentTerm {
                rf.currentTerm = reply.Term
                rf.state = "follower"
                rf.votedFor = -1
                return
            }
            if reply.VoteGranted {
                votes++
                if votes >= len(rf.peers)/2+1 {
                    rf.becomeLeader()
                }
            }
        }(i)
    }
}
```

Three things worth understanding here:

- You snapshot `term` before releasing the lock and entering goroutines. By the time a reply comes back, `rf.currentTerm` might have changed — you compare against the snapshot to detect stale replies.
- Each RPC is sent in its own goroutine so they fire in parallel, not sequentially.
- `votes` is a local variable captured by the goroutine closures — no lock needed for it since each goroutine only increments once and Go's closure captures the pointer correctly... actually, do make `votes` an `int32` and use `atomic` or just protect it with the mutex you already hold when incrementing. The version above holds `rf.mu` when incrementing `votes` so it's safe.

## Step 2 — Implement becomeLeader() (10 min)

```go
func (rf *Raft) becomeLeader() {
    rf.state = "leader"
    // Initialize leader-only state
    rf.nextIndex = make([]int, len(rf.peers))
    rf.matchIndex = make([]int, len(rf.peers))
    for i := range rf.peers {
        rf.nextIndex[i] = len(rf.log)
        rf.matchIndex[i] = 0
    }
    go rf.heartbeatLoop()
}
```

## Step 3 — Heartbeat loop (15 min)

The leader must send heartbeats regularly or followers will time out and start elections. Heartbeats are just empty `AppendEntries` RPCs for now:

```go
func (rf *Raft) heartbeatLoop() {
    for !rf.killed() {
        rf.mu.Lock()
        if rf.state != "leader" {
            rf.mu.Unlock()
            return
        }
        rf.mu.Unlock()

        rf.sendHeartbeats()
        time.Sleep(100 * time.Millisecond)
    }
}

func (rf *Raft) sendHeartbeats() {
    rf.mu.Lock()
    term := rf.currentTerm
    rf.mu.Unlock()

    for i := range rf.peers {
        if i == rf.me {
            continue
        }
        go func(peer int) {
            args := &AppendEntriesArgs{
                Term:     term,
                LeaderId: rf.me,
            }
            reply := &AppendEntriesReply{}
            rf.peers[peer].Call("Raft.AppendEntries", args, reply)

            rf.mu.Lock()
            defer rf.mu.Unlock()
            if reply.Term > rf.currentTerm {
                rf.currentTerm = reply.Term
                rf.state = "follower"
                rf.votedFor = -1
            }
        }(i)
    }
}
```

## Step 4 — AppendEntries RPC types and handler (15 min)

```go
type AppendEntriesArgs struct {
    Term     int
    LeaderId int
    // More fields come in 2B
}

type AppendEntriesReply struct {
    Term    int
    Success bool
}

func (rf *Raft) AppendEntries(args *AppendEntriesArgs, reply *AppendEntriesReply) {
    rf.mu.Lock()
    defer rf.mu.Unlock()

    reply.Term = rf.currentTerm
    reply.Success = false

    if args.Term < rf.currentTerm {
        return // stale leader
    }

    // Valid heartbeat — reset election timer
    rf.lastHeartbeat = time.Now()
    rf.state = "follower"
    rf.currentTerm = args.Term
    reply.Success = true
}
```

## Step 5 — Implement RequestVote correctly (20 min)

Replace your stub. Every check maps directly to Figure 2:

```go
func (rf *Raft) RequestVote(args *RequestVoteArgs, reply *RequestVoteReply) {
    rf.mu.Lock()
    defer rf.mu.Unlock()

    reply.Term = rf.currentTerm
    reply.VoteGranted = false

    // Rule 1: candidate's term is stale — reject
    if args.Term < rf.currentTerm {
        return
    }

    // Rule 2: higher term seen — update and step down
    if args.Term > rf.currentTerm {
        rf.currentTerm = args.Term
        rf.state = "follower"
        rf.votedFor = -1
    }

    // Rule 3: already voted for someone else this term — reject
    if rf.votedFor != -1 && rf.votedFor != args.CandidateId {
        return
    }

    // Rule 4: candidate's log must be at least as up-to-date as ours
    lastIndex := len(rf.log) - 1
    lastTerm := rf.log[lastIndex].Term
    logOk := args.LastLogTerm > lastTerm ||
        (args.LastLogTerm == lastTerm && args.LastLogIndex >= lastIndex)
    if !logOk {
        return
    }

    // Grant vote
    rf.votedFor = args.CandidateId
    rf.lastHeartbeat = time.Now() // reset timer so we don't start election
    reply.VoteGranted = true
}
```
