Before we start coding, let me explain what 3D actually is conceptually, because the implementation will make much more sense.

---

## What is Log Compaction?

Right now your log grows forever. Every command ever submitted stays in the log. After weeks of operation that's a problem:

```
log: [S, A, B, C, D, E, F, G, H, I, J, K ...]
                                              ↑ keeps growing
```

A snapshot says: *"everything up to index N is summarized in this blob — throw away the log entries before N."*

```
snapshot: {state at index 7}   log: [H, I, J, K ...]
```

The state machine (KV store, cronjob scheduler, etc.) takes a snapshot of its current state. Raft stores that snapshot and trims the log.

---

## What breaks

When you trim the log, **index 0 is no longer the sentinel entry.** Everything that assumes `rf.log[0]` exists breaks. Every index calculation needs an offset.

You need two new fields:

```go
lastIncludedIndex int  // last log index included in the snapshot
lastIncludedTerm  int  // term of that entry
```

From now on, to convert between **absolute index** (what the outside world sees) and **array index** (position in your log slice):

```
array index = absolute index - lastIncludedIndex
```

## Step 2 — Add index helpers (10 min)

These two helpers will save you from arithmetic bugs everywhere. Add them now and use them consistently:

```go
// absolute index → array position
func (rf *Raft) logIndex(absoluteIndex int) int {
    return absoluteIndex - rf.lastIncludedIndex
}

// last absolute index in log
func (rf *Raft) lastLogIndex() int {
    return rf.lastIncludedIndex + len(rf.log) - 1
}

// term at absolute index
func (rf *Raft) logTerm(absoluteIndex int) int {
    if absoluteIndex == rf.lastIncludedIndex {
        return rf.lastIncludedTerm
    }
    return rf.log[rf.logIndex(absoluteIndex)].Term
}
```

## Step 3 — Update `persist()` and `readPersist()` (10 min)

Snapshots introduce two new persistent fields:

```go
func (rf *Raft) persist() {
    w := new(bytes.Buffer)
    e := labgob.NewEncoder(w)
    e.Encode(rf.currentTerm)
    e.Encode(rf.votedFor)
    e.Encode(rf.log)
    e.Encode(rf.lastIncludedIndex)  // ← add
    e.Encode(rf.lastIncludedTerm)   // ← add
    raftstate := w.Bytes()
    rf.persister.Save(raftstate, nil)
}
```

```go
func (rf *Raft) readPersist(data []byte) {
    if data == nil || len(data) < 1 {
        return
    }
    r := bytes.NewBuffer(data)
    d := labgob.NewDecoder(r)
    var currentTerm int
    var votedFor int
    var log []LogEntry
    var lastIncludedIndex int  // ← add
    var lastIncludedTerm int   // ← add
    if d.Decode(&currentTerm) != nil ||
        d.Decode(&votedFor) != nil ||
        d.Decode(&log) != nil ||
        d.Decode(&lastIncludedIndex) != nil ||  // ← add
        d.Decode(&lastIncludedTerm) != nil {    // ← add
        return
    }
    rf.currentTerm = currentTerm
    rf.votedFor = votedFor
    rf.log = log
    rf.lastIncludedIndex = lastIncludedIndex  // ← add
    rf.lastIncludedTerm = lastIncludedTerm    // ← add
}
```

## Step 4 — Implement `Snapshot()` (15 min)

This is called by the state machine when it wants to take a snapshot:

```go
func (rf *Raft) Snapshot(index int, snapshot []byte) {
    rf.mu.Lock()
    defer rf.mu.Unlock()

    // Ignore stale or already-snapshotted index
    if index <= rf.lastIncludedIndex {
        return
    }

    // The term of the last included entry
    newLastIncludedTerm := rf.logTerm(index)

    // Trim the log — keep entries after index
    // log[0] becomes a dummy sentinel for the new base
    rf.log = rf.log[rf.logIndex(index):]
    rf.log[0].Command = nil  // sentinel — term is preserved, command not needed

    rf.lastIncludedIndex = index
    rf.lastIncludedTerm = newLastIncludedTerm

    // Persist both raft state and the snapshot
    rf.persistWithSnapshot(snapshot)
}
```

Add a helper that saves both together:

```go
func (rf *Raft) persistWithSnapshot(snapshot []byte) {
    w := new(bytes.Buffer)
    e := labgob.NewEncoder(w)
    e.Encode(rf.currentTerm)
    e.Encode(rf.votedFor)
    e.Encode(rf.log)
    e.Encode(rf.lastIncludedIndex)
    e.Encode(rf.lastIncludedTerm)
    raftstate := w.Bytes()
    rf.persister.Save(raftstate, snapshot)
}
```

And update your existing `persist()` to use it:

```go
func (rf *Raft) persist() {
    rf.persistWithSnapshot(rf.persister.ReadSnapshot())
}
```
